# 首页模块设计文档

## 1. 模块概述

### 1.1 功能描述
首页是用户访问网站的第一个页面，用于展示个人简介、最新博客、精选作品等信息。

### 1.2 模块位置
- 路由路径：/
- 页面文件：frontend/pages/index.js
- 开发环境：本地直接运行（支持热重载）

## 2. 页面设计

### 2.1 页面元素
- 个人头像和基本信息展示
- 个人简介内容（支持Markdown格式）
- 最新博客文章预览（显示3-5篇）
- 精选作品展示（显示3-5个）
- 社交媒体链接
- 网站导航菜单

### 2.2 视觉设计
- 采用深灰色背景和白色字体的对比设计
- 个人头像使用圆形设计，带有微妙的阴影效果
- 导航菜单使用浅绿色链接，hover时有颜色过渡效果
- 博客和作品预览卡片采用简约的卡片设计，带有科技感的边框
- 社交媒体图标使用浅绿色，保持整体视觉统一

### 2.3 交互设计
- 响应式布局，适配不同设备屏幕
- 点击博客标题跳转到博客详情页
- 点击作品图片跳转到作品详情页
- 导航菜单支持移动端折叠显示
- 悬停效果：卡片轻微上浮，链接颜色过渡动画

## 3. 功能实现

### 3.1 数据获取
- 从后端API获取个人基本信息
- 获取最新的3-5篇博客文章
- 获取精选的3-5个作品
- 开发环境中直接从本地运行的API服务获取数据

### 3.2 接口调用
- GET /api/user/profile - 获取用户基本信息
- GET /api/blog/posts?limit=5 - 获取最新博客
- GET /api/portfolio/items?limit=5 - 获取精选作品

### 3.3 状态管理
- 使用React Context管理用户信息状态
- 使用SWR进行数据缓存和自动重新验证

## 4. 性能优化

### 4.1 加载优化
- 图片懒加载
- 组件代码分割
- 骨架屏加载效果

### 4.2 缓存策略
- 首页数据缓存1小时
- 个人简介内容静态化
- 社交媒体链接缓存24小时

## 5. SEO优化

### 5.1 元数据
- 页面标题：个人网站名称
- 页面描述：个人简介摘要
- 关键词：个人品牌关键词

### 5.2 结构化数据
- 个人信息结构化数据
- 网站导航结构化数据
